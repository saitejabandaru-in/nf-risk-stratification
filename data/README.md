# Data Directory

Place your anonymized clinical dataset here as:

```text
data/nf_clinical_data.csv
```

Required columns are:

* `HBA1C`
* `ALBUMINA`

Optional columns used for richer ranking and reporting are:

* `MORTO`
* `AMPUTAZIONE`
* `PATIENT_ID`

The original patient-level dataset is intentionally not included for privacy
and ethics reasons.
